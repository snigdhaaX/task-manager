# Linked List Task Manager 

CLI Task Manager built with **Singly Linked List** in C++.

## Demo
=== Task Manager ===

1. Add Task
2. Remove Task
3. Mark Complete
4. Display Tasks
5. Exit
   Choice: 1
   Task: Go through a lecture
   Added.
Tasks(1 total):
  Go through a lecture YAY DONE!

## Features
- ✅ **O(1)** Add task (head insertion)
- ✅ **O(n)** Remove + search
- ✅ File persistence (`tasks.txt`)
- ✅ Proper memory management

## Build & Run
```bash
g++ task.cpp -o task
./task
